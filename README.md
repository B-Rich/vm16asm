## Installation

Download the file `vm16asm-1.0.3-py3-none-any.whl` from: https://github.com/joe7575/vm16asm/blob/main/dist/

Install it with Python `pip`, e.g. for Ubuntu:

```
sudo pip3 install vm16asm-1.0.3-py3-none-any.whl
```


## Using

Syntax:

```
vm16asm <asm-file>
```

Example:

```
vm16asm test.asm
```
